## Features

### Pre-trade validation (`validateTrade`)
- Rejects trades exceeding `maxTradeSize`
- Rejects trades when `maxPositions` is at limit
- Rejects trades for symbols not in `symbolWhitelist` (empty = all allowed)
- Rejects trades when kill switch is triggered
- Rejects trades that would cause committed capital to exceed `maxCapital`
- Rejects trades when `maxDailyTrades` is reached
- Rejects trades when `maxDailyLoss` is reached
- All rejection reasons are human-readable strings

### Kill switch
- Auto-triggers in `onEquityUpdate` when drawdown from peak equity exceeds `killSwitchDDPercent`
- Remains triggered after equity recovers (requires manual `resetKillSwitch()` call)
- Can be disabled via `killSwitchEnabled: false` in config
- Records `killSwitchTriggeredAt` Unix ms timestamp on trigger
- `checkKillSwitch()` returns drawdown-based reason message at all times

### Equity and position tracking
- High water mark (`peakEquity`) tracked via `onEquityUpdate`
- `currentDrawdownPercent` = `(peak - current) / peak * 100`
- `openPositionCount` increments on `onTradeOpened`, decrements on `onTradeClosed` (never below 0)
- `committedCapital` accumulates on `onTradeOpened` for capital ceiling checks

### Daily counters
- `dailyTradeCount` increments on `onTradeOpened`
- `dailyLoss` accumulates magnitude of losses (profits excluded) on `onTradeClosed`
- `resetDailyCounters()` resets both to 0 and updates `lastResetDate` (YYYY-MM-DD)

### Config management
- Constructor validates all config fields and throws descriptive errors on invalid values
- `updateConfig(partial)` validates the merged result before applying
- `getConfig()` returns a defensive copy

## Config Interface

```typescript
interface RiskManagerConfig {
  maxCapital: number;           // Max total committed capital
  maxTradeSize: number;         // Max single trade size
  maxPositions: number;         // Max simultaneous positions
  killSwitchEnabled: boolean;   // Toggle kill switch on/off
  killSwitchDDPercent: number;  // Drawdown % to trigger kill switch (0,100)
  symbolWhitelist: string[];    // Allowed symbols (empty = all)
  maxDailyLoss?: number;        // Optional: max daily loss in $
  maxDailyTrades?: number;      // Optional: max trades per day
}
```
